# Student Degree Outcome Prediction and Pattern Mining

Streamlit deployment package for student stop, shift, and completion prediction using enrollment/profile data.

## Main file
`app.py`

## Deployment steps
1. Upload this repository to GitHub.
2. In Streamlit/AuthKit deployment, choose `app.py` as the entry file.
3. The app accepts an Excel upload at runtime. Do not upload confidential student data to a public GitHub repository.

## Required files
- app.py
- requirements.txt
- runtime.txt
- packages.txt

## Optional
Place a default Excel file in `data/StudProfile.xlsx` only if the repository is private.
