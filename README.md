# Student Degree Outcome Prediction Streamlit App

## Deploy steps
1. Upload these files to the root of a GitHub repository.
2. Make sure `app.py`, `requirements.txt`, `runtime.txt`, and `packages.txt` are in the repository root.
3. Deploy on Streamlit/AuthKit and set the main file path to `app.py`.
4. Upload the Excel dataset inside the app at runtime.

## Important
Do not upload private student data to a public GitHub repository. Use the app uploader instead.
